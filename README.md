# 2048 Python Game (Console Version)

This is a simple implementation of the classic **2048 Game**, created using **pure Python** without any external libraries such as Pygame.  
The game runs directly inside your **terminal / command prompt**.

How to Play

- Board size: **4 × 4**
- Use the following keys:
  - **W** → move up
  - **A** → move left
  - **S** → move down
  - **D** → move right
  - **Q** → quit the game

- When two tiles with the same number collide, they merge.
- New tiles (2 or 4) appear after every move.
- Goal: **Reach 2048!**
